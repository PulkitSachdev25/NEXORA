Project Title - 
The Keating Scale

Description - 
The Keating Scale is a web based tool designed to channel one's inner creativity. In an age where everyone is moving towards AI automation, orignality and uniqueness are becoming rare.
The tool finds its inspiration from the movie 'Dead Poets Society'. In the movie, teacher Mr. John Keating lays emphasis on the importance of expanding one's ability to think for ourselves. 

Problem Statement - 
The problem faced was the rise of repetitive and generic communication. The increase in writing emails , essays and even social media posts using AI has led to the loss of orignality and the human essence of ideas.
People often rely on overused phrases, motivational clichés, or vague statements instead of expressing personal, grounded thoughts.

This leads to:

loss of authentic voice in writing
shallow expression of ideas
reduced emotional depth in communication
dependence on AI-generated or templated language

Solution - 

The solution was to create a tool which doesn't correct the user, but instead it pushes the user to think for themselves. This way, one can put forward their points more freely and openly. 
The Keating Scale analyzes user-written text using five core dimensions:

Authenticity

Originality

Emotional Weight

Clarity

Boldness

Each input is processed and assigned structured scores along with a qualitative interpretation.

Instead of rewriting the text, the system provides a Keating Push, which highlights what is missing and gives constraints that force the user to rethink and rewrite their own expression.

The system works as a reflective loop:
User writes → System analyzes → System critiques → User improves writing themselves.

This ensures the user remains the author of their expression rather than the AI replacing it.
